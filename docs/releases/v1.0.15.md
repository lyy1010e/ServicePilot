# ServicePilot 1.0.15

## 新功能与改进

- 新增“启动时恢复上次正常退出前仍在运行的服务”设置，默认关闭；恢复清单独立保存并只消费一次。
- 点击更新时会重新检查并下载当前最新的已验证版本，避免使用旧的更新信息。
- 更新提示改为居中展示，只提示有新版本，不再展示冗长更新内容。
- 优化服务列表勾选框对齐和运行时长刷新机制。

## 稳定性修复

- 修复服务重启时旧进程退出回调可能覆盖新进程状态的问题。
- Windows 下将受管服务及启动辅助进程纳入 Job Object；ServicePilot 异常退出时会清理受管进程树，避免遗留进程占用端口。
- 日志历史改为维护字节计数，仅在达到总量上限时批量回收，降低高频日志下的持续开销。
- 对配置或识别到本地端口的服务增加后台连通性巡检；连续失败会显示告警，恢复后自动清除，不会自动停止服务。

## 验证

- TypeScript 静态检查、前端单元测试和 Rust 单元测试均已通过。
