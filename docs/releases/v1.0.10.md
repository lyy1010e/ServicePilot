# ServicePilot 1.0.10

## 新增

- 日志搜索改为类似 IDEA 的定位体验：搜索时保留完整日志上下文，只高亮命中内容，并支持上一条/下一条跳转到匹配位置。
- 新建或导入 Spring 服务时默认使用本地开发 JVM 内存参数 `-Xms128m -Xmx512m`，降低多服务联调时的内存压力。
- 新增 `npm run check:rust`，用于快速检查 Tauri/Rust 后端代码，无需完整启动桌面应用。

## 优化

- 统一弹窗主按钮与应用内其它按钮的视觉风格，减少蓝色按钮样式不一致的问题。
- ServicePilot 自身更新切换为静默安装模式，减少 Windows 安装进度弹窗干扰。
- 前后端模块边界继续收敛，服务、分组、日志、设置等领域逻辑更清晰。

## 修复

- 修复日志搜索只显示匹配行导致无法查看上下文的问题。
- 改进 Java Main / Maven 预设的默认 JVM 参数处理，避免覆盖已有服务的手动配置。
